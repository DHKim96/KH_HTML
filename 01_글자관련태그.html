h태그는 상당히 많이 사용한다.

문단을 나누는 태그 : p, pre

문단 영역을 나누는 태그로는 p태그와 pre태그가 있다. p태그는 줄바꿈을 하고자 한다면 별도의 태그를 작성해야한다.
그리고 공백은 한 개의 공백만을 표시하기 때문에 별도의       기호 문구를 기술해야한다.

p태그는 많이 사용 

pre태그는 시작 태그에서부터 종료 태그까지 입력된 내용을 그대로 표현하는 태그다
줄바꿈 이라든가          여러 개의 공백을 바로 인식함
양식 받았는데 너무 길어서 그대로 쓰고자 할 때 사용하낟

그 밖의 글자를 다루는 태그들

일반글꼴(태그로 감싸지 않은 텍스트)

b : 글자를 굵게 표시하는 태그

strong : 글자를 굵게 표시하는 태그 + 스크린리더(시각장애인을 위한 SW)

em : 글자를 기울여서 보여주는 태그

i : 글자를 기울여 표시하는 태그

mark : 형광펜 효과를 주는 태그

u : 글자에 밑줄이 그어지는 태그

s : 글자에 취소선을 넣어주는 태그

small : 글자를 작게 표현해주는 태그

기본 글자에 sub : 아래 첨자를 나타내는 태그와 sup : 윗첨자를 나타내는 태그
